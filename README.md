# Andrew Hui 火炭 Personal Training Landing Page

一個為 **Andrew Hui** 製作的靜態宣傳頁面，主打火炭銀禧會的 1 對 1 增肌、減脂、塑形與體驗課程預約。

## 專案內容

此專案包含：

- `index.html`：主頁面結構與互動邏輯
- `style.css`：視覺樣式與版面設計
- `content.json`：文案內容設定
- `images/`：圖片素材

## 使用方式

### 本機預覽
直接用瀏覽器打開 `index.html` 即可預覽。

### 更新內容
如需修改頁面文案、表單標題或 CTA，請優先編輯：

- `content.json`

如需調整版面或互動效果，請編輯：

- `index.html`
- `style.css`
